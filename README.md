# Maa Vindhavashani - Taxi Service Website

Professional website for Maa Vindhavashani car taxi service in Bettiah, Bihar.

## Features

- ✨ Modern, responsive design
- 🚕 7-seater Ertiga taxi service showcase
- 📱 Mobile-friendly interface
- 🎨 Built with React, Vite, and Tailwind CSS
- 🚀 Fast performance with Vite
- 📞 Contact form and multiple contact options
- 💳 Pricing information
- 📍 Location-based services

## Project Structure

```
├── index.html              # HTML entry point
├── package.json            # Dependencies
├── vite.config.js         # Vite configuration
├── tailwind.config.js     # Tailwind CSS configuration
├── postcss.config.js      # PostCSS configuration
├── src/
│   ├── main.jsx           # React entry point
│   ├── App.jsx            # Main app component
│   └── index.css          # Global styles
└── maa-vindhavashani-website.jsx  # Main website component
```

## Installation

1. **Clone or extract the project**

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```
   The website will open at `http://localhost:3000`

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```

## Technologies Used

- **React** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Icon library
- **PostCSS** - CSS processing

## Customization

### Colors
Edit `tailwind.config.js` to change the color scheme:
- Blue (#1e3a8a) - Primary color
- Yellow (#fbbf24) - Accent color

### Content
All text content can be easily modified in the `maa-vindhavashani-website.jsx` file.

### Images
Replace emoji icons with actual images by modifying the service cards and feature sections.

## Deployment

- Any static hosting service
